# Billing System

### Setup

Once your cloned the repository, make sure your system have python installed, then just run the below automation files do install requirements and start the project using below command.

`> python install_requirements.py`

Once successfully this automation code installed frontend and backend packages, then run start file using below command, please make sure, there is no other project is running in localhost:3000 and localhost:8000 ports, because this project requires those.

`> python start.py`

Once you run the wait for 3 to 4 mins for react ui start, your default browser automatically have tab with localhost:3000 to show the project output
